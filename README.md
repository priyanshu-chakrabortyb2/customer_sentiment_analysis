# customer_sentiment_analysis
## Overview

An end-to-end NLP and Machine Learning project that analyzes customer reviews and predicts sentiment using TF-IDF and Linear SVC.

## Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* TF-IDF Feature Engineering
* Sentiment Classification
* Hyperparameter Tuning
* FastAPI REST API
* SQLite Database Integration
* Interactive Streamlit Dashboard
* Batch CSV Prediction
* Dockerized Deployment

## Dataset

Amazon Customer Reviews Dataset

## Machine Learning Models

* Multinomial Naive Bayes
* Logistic Regression
* Linear SVC (Final Model)

## Model Performance

| Model               | Accuracy |
| ------------------- | -------- |
| MultinomialNB       | 87.7%    |
| Logistic Regression | 84%      |
| Linear SVC          | 88%      |

Final Model: **Linear SVC**

## Technology Stack

* Python
* Pandas
* NumPy
* Scikit-Learn
* NLTK
* FastAPI
* SQLite
* Streamlit
* Plotly
* Docker

## Project Workflow

Data Cleaning → Text Preprocessing → TF-IDF → Model Training → Hyperparameter Tuning → Model Evaluation → FastAPI → SQLite → Streamlit Dashboard
